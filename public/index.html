A physics-based puzzle game that transforms your phone into a motion-controlled playground! Tilt your device to guide the ball through challenging mazes and clear the screen.

Key Features:
  • Motion-controlled gameplay using device sensors
  • Dynamic level system powered by Firebase
  • Real-time level updates without app updates
  • Global leaderboard system
 Built with:
  • Native Android (Kotlin)
  • Firebase Remote Config + Firestore
  • Authentication with Play Games
  • Device Motion Sensors
  • Google AdMob Integration
